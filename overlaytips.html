What Can I Do?

Options for Overlays In Style

Rules - the site states that images cannot be misleading about the actual pet or stats. The line here is what can actually be achieved, and what's just for fun.

If you COULD have that image (pet, item, paint job), but you don't, that's a safe bet to avoid it.

Examples of Don'ts

  • You cannot overlay your Slorg petpet with a Meowclops. Users can get a Meowclops, so this is misleading.
  • Overlaying your red pet with a Halloween paint job. (I imagine a worst-case scenario with someone arranging a pet trade with a fake painted pet)
  • Replacing your user shield with a shield that lists a different account age.

Examples of OK

  • Belongings are apparently ok to play around with.
  • Decorative overlays that aren't imitating another pet that exists (e.g. I have the actual Great A'Tuin from Discworld as the image on GreatATuin's pet lookup. I have seen people with Pokemon name pets using the actual pokemon image.)
  • Similarly, user shields with NO account age seem to be ok.

Takeaway: Just a pretty pic? OK. A pic of something on NPC that you don't actually have there? Not OK.

HTML Elements

Each HTML element is a piece of the page content, like a paragraph, or a header, or an image. Sometimes it's a container like a div or a table.

You need to be able to select elements to style them.

CSS Selectors

Selectors name the elements on the page you're trying to edit. The selector is named first, and then all your style rules for that element get put inside curly braces.

Below I am selecting the header of the page.

header {width: 200px;}

CSS Properties

A property is any characteristic you're trying to change about the element. Common properties include width or font color. In the example above, the width property is in pink. The value is 500px.

Multiple properties can be changed within the same curly braces. Each property and value gets followed by a semicolon, including the last one!

Below, properties are in pink, and their values are in yellow.

header {width: 200px;
        background-color: goldenrod;
        font-family: sans-serif;}

Remove Background

Transparent images are the golden child of lookup edits, especially for pets and belongings. Unfortunately, you have to make your own transparent image and host it for every single thing you want to change.

The good news is that there's an online tool that has been around a while and that I find really consistent in removing backgrounds to make an image transparent: remove.bg.

Paste an image into remove.bg to erase its background. (This site has been around since well before Midjourney, and has terms I find acceptable in this day and age.)

You still need to copy and save the now-transparent image, then host it somewhere in order to replace any NPC images with it. You can use the content:url code above to do that, linking your own image.
