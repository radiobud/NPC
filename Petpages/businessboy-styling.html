CSS tips for NPC Styling

Using selectors to work with site code

Most of the Neo elements use IDs or classes, so you need to select those in your CSS. How to do this?

Dev Tools

When you are trying to identify an element to style, Developer Tools (Dev Tools) are your friend.

You can access them with F12, or - my favorite way - hit CTRL+SHIFT+C on a PC. This will let you hover around the page and highlight a specific element to select.

Here's an example in action

You can see the name, class, or ID of the element, which is useful for trying to style it. This element is called .petStatsBlock !

The inspector also opens Dev Tools, which looks like this - notice the stat block is highlighted.

If you click the element, it stays in focus, and you can edit the CSS within the Dev Tools and see live what it does.

Try it out?

Go to one of your own petpages, and open the Element Inspector by pressing CTRL+SHIFT+C (or CMD+OPT+C on mac). Then click on the stats block.

The Styles window should show up in your dev tools. Try adding a new background-color to the block!

I'm changing Watercolour's to lavender.
