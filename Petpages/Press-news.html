Coding Tips of the Week

Overlays for Anything

Today's tip is about how to place an overlay for whatever you want. The basic code is below:

.someThing img {
    content: url("overlayURLhere");
    }

What does this code mean? It is saying that for the "someThing" class, any image will have the content you give it. And the content you give is in the URL in parentheses. You could make every pet on your lookup have the image of a Slorg. (Except I think you are not supposed to use misleading overlays for key things like pet species).

Troubleshooting Tuesday

Check your Code

Most of the time when code does not work, it's either a mis-used property or it's a plain old typo. Guess which one is easier to troubleshoot?

A misplaced bracket or a missing semicolon will completely break a lot of code. If things become bad fast, check for punctuation.

Remember to put a semicolon after every property in CSS.
