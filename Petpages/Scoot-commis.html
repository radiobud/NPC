What to expect

I am interested in getting more people comfortable coding.

Code that I write for you will have plentiful comments throughout, explaining important sections and pointing out things you can customize.

In general, I'll write CSS that you'll want to keep inside style tags, like these:

<style>

</style>

The comments in the CSS style section will be placed between slashes and asterisks. These symbols keep anything between them from being read by the browser.

(Comments are also a great way to remove something temporarily without deleting it. You might call it "commenting out" a line of code.)

/* Here's a CSS comment. */

/* Below is a line of code. */

    header{display:none;}

/* If you don't want the code to take effect,
comment it out like below: */

    /* header{display:none;} */

/* Missing punctuation or un-closed comments
can mess with your CSS, so be careful. */

Take a look at the Val's CSS Intro page for refresh of CSS basics.

Another quick note

I will not be able to do graphic editing for you, beyond simple removal of backgrounds to make transparent images.

I enjoy digital painting and design but am not up for that level of involvement in the creation of a lookup yet.

That said, if you have an aesthetic or a character/fandom look in mind, I'm happy to find existing images or use your chosen images in the layout.

You do the dreaming and I'll make the structure and style for it!
